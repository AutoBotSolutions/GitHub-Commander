# Keyboard Shortcuts

This guide lists all keyboard shortcuts available in GitHub Commander for power users.

## Global Shortcuts

| Action | Shortcut | Description |
|--------|----------|-------------|
| New Repository | `Ctrl+N` | Create a new repository |
| Settings | `Ctrl+,` | Open settings dialog |
| Refresh | `F5` | Refresh current view |
| Quit | `Ctrl+Q` | Quit application |

## Customizing Shortcuts

Currently, GitHub Commander does not support custom keyboard shortcuts. This feature may be added in future versions.

## Tips for Using Shortcuts

1. **Memorize frequently used shortcuts** - Focus on the ones you use most
2. **Use navigation shortcuts** - Alt+number keys for quick panel switching
3. **Combine with mouse** - Use shortcuts for common actions, mouse for navigation
4. **Practice makes perfect** - Regular use builds muscle memory

## Missing Shortcuts

If you find a shortcut missing that would be useful:
1. Check if the action exists in the menu
2. Consider opening an issue on GitHub
3. The feature may be added in future releases

## Platform Differences

These shortcuts are designed for Linux. On other platforms:
- **macOS**: Replace `Ctrl` with `Cmd` (⌘)
- **Windows**: Shortcuts should work as listed

Note: GitHub Commander is currently Linux-only.
